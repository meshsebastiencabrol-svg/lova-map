Step 3 of 8
Get a Google Maps API Key
This is the key that makes the map work — takes about 5 minutes
1
Go to console.cloud.google.com and sign in with your Google account
2
Click "Select a project" at the top → click "New Project" → name it anything (e.g. Lova Map) → click Create
3
Go to console.cloud.google.com/billing → click "Link a billing account" → enter your credit card. Google gives $200 free credit every month — you will not be charged for normal use.
4
Go to console.cloud.google.com/google/maps-apis/credentials → click "+ Create Credentials" → select "API key"
5
A popup shows your key — it looks like AIzaSyB3xxxxxxxxxxxxxxx — click Copy and save it somewhere safe
⚠️ Keep your API key private — do not share it publicly or post it on social media. Anyone with your key could use your Google Maps quota.

Step 5 of 8
Generate the Map File with Claude AI
Ask Claude to build the map — it takes one conversation
1
Go to claude.ai and sign in (free account works)
2
Start a new conversation and paste this prompt:
Create an interactive HTML map of Australia using Google Maps API that shows:
- Blue pins for all Harvey Norman stores
- Red pins for all The Good Guys stores
- Orange pins for all JB Hi-Fi stores
- Green pins and 200km radius circles for Brand Ambassadors

Include toggle buttons to show/hide each layer, a radius slider,
and a click panel showing store name and details.
Hardcode the API key directly into the file.
The map should load instantly with no searching.
3
Follow Claude's instructions — it will ask for your BA list (names + suburb + state) and will build the complete HTML file
4
When Claude gives you the file, download it. Then ask Claude to "hardcode my API key into the file" and paste your API key — this means the map opens automatically without anyone needing to enter a key
5
To get accurate store coordinates, use the built-in ⚙️ Update Store Data button inside the map — it geocodes every store automatically using your API key
💡 Tip: The more detail you give Claude about your BAs (name, suburb, state) the more accurate the map will be. You can also upload a spreadsheet directly to Claude.

Step 6 of 8
Update Store Coordinates
Use the built-in geocoder to get accurate locations for every store
The map file includes a built-in ⚙️ Update Store Data tool that automatically looks up GPS coordinates for all 500+ stores using Google's Geocoding API.
1
Open your lova_sharkninja_map.html file in your browser (double-click it)
2
Click the ⚙️ Update Store Data button in the top right of the header
3
Paste your Google Maps API key and click "Find All Store Coordinates →"
4
Wait 3–5 minutes while it geocodes all stores — you'll see a progress bar
5
Click 📋 Copy All and paste the result back into Claude — Claude will update the map file with all the accurate coordinates
⏱ This step takes about 5 minutes but only needs to be done once. After that the map loads instantly every time.

Step 7 of 8
Host on GitHub Pages
Get a permanent shareable link — free forever, takes 5 minutes
1
Go to github.com → sign up for a free account (or log in if you have one)
2
Click the green "New" button → name the repository lova-map → set to Public → tick "Add a README file" → click Create repository
3
Inside the repository click "Add file" → "Upload files" → drag your lova_sharkninja_map.html file → click Commit changes
4
Click Settings (top menu) → scroll left sidebar to Pages → under Source select "Deploy from a branch" → choose main → click Save
5
Wait 2–3 minutes then refresh — you'll see a green banner: "Your site is live at..."
Your permanent link will be:

https://YOUR-USERNAME.github.io/lova-map/lova_sharkninja_map.html
